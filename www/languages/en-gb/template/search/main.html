Project Search Finds Projects in Organisation Units via Name and/or Template. File/Folder/Value Search Finds Files, Values or Folders in Folders via Name or Extension. User/Group Search Finds Users or Groups.
Sample Search Finds Samples in Organisation Units via Name, ID and/or Template. Data Search Finds Data in Projects or Samples uncoupled to Folder-structure. Full Text Search Makes a Full Text Search over all Values with Text, including Project Descriptions.
